提交本地代码

首先使用add命令把想要提交的代码添加进来，然后用commit命令执行提交操作。常见的命令及作用：

git add 文件名. 添加单个文件

git add 目录名. 添加一个目录下的所有文件

git add .. 添加所有的文件

git commit -m "描述信息". 提交添加的文件

忽略文件

忽略文件是名为.gitignore的文件，在..gitignore文件中每一行指定的文件就不会加入到版本控制当中。

与修改内容有关的一些命令及作用

git status. 查看文件修改情况

git diff. 查看文件的更改内容

git diff 文件路径. 查看一个文件的更改内容

git checkout 文件路径. 撤销对一个文件的修改（注意：只能撤销未提交的文件的修改）

git reset HEAD 文件路径. 取消文件的添加

（对于已提交的文件，可以先用此命令取消添加，然后再用git checkout撤销修改）

git log. 查看所有历史提交信息

git log 一记录的id -1. 只查看一条记录

git log 一记录的id -1 -p. 查看一条提交记录具体修改了什么内容
